Help

Functions

  • shape(center, color, sideNum, sideLength)
    Usage: Draw any polygon of side number, sideNum with radius, sideLength around a point [x, y] and a color
  • drawLine(sx, sy, angle, linelength, mcolor)
    Usage: Draws a line from point sx,sy at an angle from 0 with a linelength and color
  • degrees( radians )
    Usage: Takes an angle in radians and returns degrees
  • radians( degrees )
    Usage: Takes an angle in degrees and returns the radians

Variables

  • transformStack = []
    Usage: Type is array, can use transformStack.push(element) and transformStack.pop()
  • currentPos = [0,0,0]
    Usage: Represents an X, Y and Angle given where the next drawing will start. Push and pop this to save progress
  • startSize = 0.1
    Usage: A useful variable to store the starting line length

Example Usage

45 Degree Binary Tree

  • Alphabet: 0,1,[,]. Axiom: 0. Rules 1:11,0:1[0]0
  • 0 or 1: drawLine(currentPos[0],currentPos[1],currentPos[2], startSize, [1,0,0,1]);currentPos[0]+= Math.cos(radians(currentPos[2]))*startSize;currentPos[1]+= Math.sin(radians(currentPos[2]))*startSize;
    Meaning: Draws a line from the current x and y at current angle theta (currentPos = [x,y,theta] with length of startSize and red, then calculates the next position of the tip and stores it in currentPos
  • [ :var copy = [currentPos[0], currentPos[1], currentPos[2]];transformStack.push(copy);currentPos[2] += 45;
    Meaning: Pushes the current x, y and theta to the stack and then increases the angle by 45
  • ] :currentPos = transformStack.pop();currentPos[2] -= 45;
    Meaning: Pops the top of the stack, aka the last saved x, y and z and sets it to the current one, then decreases angle by 45

Barnsley Fern

  • Alphabet: X,F,+,-,[,] Axiom: X Rules:X:F+[[X]-X]-F[-FX]+X,F:FF
  • F :drawLine(currentPos[0],currentPos[1],currentPos[2], startSize, [250/255,100/255,60/255,1]);currentPos[0]+= Math.cos(radians(currentPos[2]))*startSize;currentPos[1]+= Math.sin(radians(currentPos[2]))*startSize;
    Meaning:Draws a line and calculates new start point
  • + :currentPos[2] -= 25;
    Meaning: Decreases current angle by 25
  • - :currentPos[2] += 25;
    Meaning: Increases current angle by 25
  • - :var copy = [currentPos[0], currentPos[1], currentPos[2]];transformStack.push(copy);
    Meaning: Pushes current position and angle
  • - :currentPos = transformStack.pop();
    Meaning: Pops current angle
